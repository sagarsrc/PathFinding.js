## Task

Build scoring and comparison features under `visual/scenario/`:

1. **Metrics capture** — capture: nodes explored, path length, time (ms) for each pathfinding run.
2. **Score card display** — show metrics for the most recent run.
3. **Save run** — save a run record (algorithm + map state + metrics + timestamp) to an in-memory store.
4. **Load saved runs** — retrieve saved runs for display.
5. **Clear all saved runs** — reset the store.
6. **Comparison drawer** — side-by-side display of two or more saved runs, showing metric differences.

Export a clean API: `captureMetrics(runData)`, `saveRun(run)`, `getSavedRuns()`, `clearRuns()`, `compareRuns(runIds)`.

## Where to build

Add to `visual/scenario/`. Do NOT modify existing `visual/js/` or `visual/index.html`.
